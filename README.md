# Superstore-KPI-Dashboard-using-Excel
This project presents a Sales KPI Dashboard built in Microsoft Excel using the Superstore dataset. The dashboard provides an overview of sales performance, regional performance, category performance, and top-selling products through interactive visualizations and KPI tracking.
## Objective

The objective of this project is to analyze sales data and identify trends, high-performing categories, top-selling products, and regional performance to support data-driven business decisions.
## Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Visualization
